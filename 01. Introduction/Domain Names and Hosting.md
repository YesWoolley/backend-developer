
# Domain Names and Hosting — Quiz

---

## 1. What is the primary purpose of the Domain Name System (DNS)?

- [ ] To provide storage space for website files and databases
- [✓] **To translate human-readable domain names into numerical IP addresses**
- [ ] To encrypt data transmission between a browser and a server
- [ ] To manage the registration of legal business entities

> **Explanation:**
> DNS acts as the phonebook of the internet, translating domain names into IP addresses because computers communicate using numbers, while humans prefer memorable names like "example.com."

---

## 2. Which component is responsible for storing the actual files, images, and code that make up a website?

- [ ] Domain Registrar
- [✓] **Web Hosting Server**
- [ ] Root Directory
- [ ] Content Delivery Network

> **Explanation:**
> Web hosting provides the physical or virtual disk space on a server where website data resides. A registrar only manages the ownership of the name, not the storage of the site content.

---

## 3. What does the TLD (Top-Level Domain) refer to in the URL "www.google.com"?

- [ ] The internal server path
- [ ] The specific sub-page of the site
- [ ] The registration protocol
- [✓] **The extension suffix ".com"**

> **Explanation:**
> The TLD is the final segment of a domain name located after the last dot. It categorizes the domain, such as .com for commercial entities, .org for organizations, or .edu for educational institutions.

---

## 4. Which type of hosting provides a website with its own dedicated physical server, offering maximum performance and control?

- [ ] Shared Hosting
- [ ] Cloud Hosting
- [✓] **Dedicated Server Hosting**
- [ ] Virtual Private Server (VPS)

> **Explanation:**
> Dedicated Server Hosting means the user rents the entire physical machine. Shared hosting divides resources among many users, and VPS uses virtualization to partition a single server into multiple isolated environments.

---

## 5. What is the main function of an A Record in a domain's DNS settings?

- [ ] To redirect one domain to another domain
- [✓] **To point a domain name to a specific IPv4 address**
- [ ] To verify ownership for email delivery
- [ ] To map a subdomain to an alias

> **Explanation:**
> An A (Address) record is the fundamental DNS record used to map a domain or subdomain to the static IPv4 address of the hosting server where the website is hosted.

---

## 6. When a user registers a domain, what does "WHOIS privacy" protect?

- [ ] The website's server speed
- [ ] The site's Search Engine Optimization rankings
- [✓] **The contact information of the domain owner**
- [ ] The server's firewall settings

> **Explanation:**
> WHOIS privacy masks the personal contact details—such as name, address, and phone number—of the domain registrant in the public WHOIS database, preventing identity theft and unsolicited spam.

---

## 7. Which type of hosting environment simulates a dedicated server by using virtualization software, but still shares the underlying physical hardware?

- [ ] Colocation
- [✓] **Virtual Private Server (VPS)**
- [ ] Reseller Hosting
- [ ] Shared Hosting

> **Explanation:**
> A VPS provides a partitioned section of a server with guaranteed dedicated resources (RAM/CPU) for a specific user, offering more stability and control than shared hosting without the cost of a full dedicated machine.

---

## 8. What is an SSL certificate primarily used for in the context of web hosting?

- [ ] Increasing the amount of storage space for media files
- [✓] **Encrypting the connection between the user's browser and the server**
- [ ] Faster indexing of website pages by search engines
- [ ] Reducing the downtime of a web server

> **Explanation:**
> An SSL (Secure Sockets Layer) certificate encrypts the data transmitted between the client and server. This is essential for protecting sensitive information, such as passwords or credit card data, from interception.

---

## 9. What does "bandwidth" refer to in a web hosting plan?

- [ ] The physical size of the website's database
- [✓] **The amount of data that can be transferred between the site and users over a given period**
- [ ] The speed at which the server processor completes tasks
- [ ] The total number of email accounts allowed on the server

> **Explanation:**
> Bandwidth represents the data volume allowed to travel to and from your website. If a site has many visitors or large files, it consumes more bandwidth, requiring a higher limit to prevent the site from going offline.

---

## 10. Which of the following is an example of a Country Code Top-Level Domain (ccTLD)?

- [ ] .net
- [✓] **.uk**
- [ ] .info
- [ ] .org

> **Explanation:**
> A ccTLD is a two-letter extension assigned to a specific country or territory. In this case, .uk represents the United Kingdom, whereas .com, .net, and .org are Generic Top-Level Domains (gTLDs).

---

## 11. What occurs if a domain name expires and the owner fails to renew it during the grace period?

- [ ] The domain is automatically transferred to the hosting provider
- [✓] **The domain enters the "Redemption Period" and may eventually be released to the public**
- [ ] The website content is permanently deleted from the internet
- [ ] The DNS records are automatically updated to point to a parked page

> **Explanation:**
> Once the grace period passes, the domain enters a redemption period where the owner can still reclaim it for a high fee. If not reclaimed, it eventually becomes available for any member of the public to purchase.

---

## 12. What is a "Subdomain"?

- [ ] An alternative extension for the same domain
- [ ] A hidden directory on the server
- [✓] **A prefix added to the primary domain (e.g., blog.example.com)**
- [ ] An IP address that points to the main site

> **Explanation:**
> A subdomain is a subset of the main domain. It allows for the creation of distinct sections of a website, such as "shop.example.com" or "support.example.com," managed under the umbrella of the main domain name.

---

## 13. In the context of hosting, what is a "control panel" (e.g., cPanel, Plesk)?

- [ ] A physical hardware switch for the server
- [✓] **A graphical interface used to manage hosting features and server settings**
- [ ] A diagnostic tool to test internet connection speed
- [ ] A security software that blocks malicious traffic

> **Explanation:**
> A control panel acts as a management dashboard, allowing users to perform administrative tasks like creating email accounts, installing CMS software, or managing databases without needing to use command-line interface commands.

---

## 14. Which hosting model involves multiple servers working together to distribute traffic and ensure high availability?

- [ ] Shared Hosting
- [ ] Dedicated Hosting
- [✓] **Cloud Hosting**
- [ ] Colocation Hosting

> **Explanation:**
> Cloud hosting utilizes a network of virtual servers that pull resources from a centralized pool. If one server hardware fails, the site can shift to another server in the network, ensuring minimal downtime.

---

## 15. What is the purpose of an MX record in a domain’s DNS zone file?

- [ ] To point the domain to the website's IP address
- [✓] **To specify the mail server responsible for receiving email for the domain**
- [ ] To prove that the domain belongs to the user
- [ ] To improve the loading speed of website images

> **Explanation:**
> MX (Mail Exchange) records are DNS entries that tell the internet where email sent to your domain name should be delivered. Without these, email providers would not know which server is handling your domain's incoming mail.
